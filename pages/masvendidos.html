Destacado de este mes

Este kit wifi autoinstalable es ideal para una colocación en el hogar. El kit trae 4 cámaras wifi con sus fuentes de alimentación y una grabadora con disco de 1TB. Las cámaras se enlazan por wifi automáticamente con la grabadora, sin necesidad de realizar cableado de video.

La grabadora se conecta a internet, y por medio de este es posible visualizar las cámaras utilizando una aplicación para el celular. También es posible visualizarla mediante la pc utilizando un software CMS. Las cámaras tienen una resolución de 1080p.
